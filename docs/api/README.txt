THƯ MỤC: docs/api/

MỤC ĐÍCH:
- Chứa tài liệu API endpoints
- Hướng dẫn sử dụng API
- Swagger/OpenAPI specifications

NỘI DUNG:
- openapi.yaml - OpenAPI specification
- endpoints.md - Danh sách endpoints
- authentication.md - Hướng dẫn xác thực
- examples.md - Ví dụ request/response
- error_codes.md - Mã lỗi API

VÍ DỤ:
- GET /api/products - Lấy danh sách sản phẩm
- POST /api/products - Tạo sản phẩm mới
- GET /api/products/{id} - Lấy sản phẩm theo ID
- PUT /api/products/{id} - Cập nhật sản phẩm
- DELETE /api/products/{id} - Xóa sản phẩm

CÔNG CỤ:
- Swagger UI (tự động từ code)
- Postman collection
- OpenAPI/Swagger documentation

GHI CHÚ:
- Tài liệu API phải cập nhật khi có endpoint mới
- Cung cấp ví dụ request/response rõ ràng
- Liệt kê tất cả lỗi có thể xảy ra
