THƯ MỤC: database/mongodb/

MỤC ĐÍCH:
- Chứa các script/config cho MongoDB
- Collection definitions
- Dữ liệu mẫu ban đầu

NỘI DUNG:
- Collections/ - Định nghĩa collections
- Indexes/ - Index definitions
- Seeds/ - Dữ liệu mẫu
- Backups/ - Backup dữ liệu

VÍ DỤ:
- narration_collection.json - Schema cho narration
- users_collection.json - Schema cho users
- initial_data.json - Dữ liệu ban đầu

GHI CHÚ:
- MongoDB không bắt buộc schema nhưng nên định nghĩa
- Backup dữ liệu quan trọng thường xuyên
- Tối ưu hóa indexes để cải thiện performance
